Contudo, você não precisa fazer um dashboard completamente novo para cada nova aplicação ou produto diferente.
É possível trabalhar com dashboards prontos que são cedidos pela comunidade, que é algo sensacional no Grafana.

Vamos imaginar que queremos um dashboard pronto para conferir a JVM. Como fazemos isso? Na lateral esquerda do Grafana, 
vamos apertar no símbolo "+" e escolher a opção "Import" onde podemos importar um JSON inteiro ou o ID de um dashboard.

No site de Dashboards do Grafana, pesquise por "Spring Boot" e você vai encontrar várias referências. 
Uma delas é Spring Boot 2.1 System Monitor. Podemos copiar o ID 11378, colocar no campo de importação do Grafana e clicar em "Load" para carregar.
